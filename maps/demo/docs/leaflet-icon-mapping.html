Leaflet 图标映射表

这份表对应当前参考层前端实现，供 references/ancient-greece-major-leaflet.geojson 使用。

前端字段优先级：

  1. properties.primaryPlaceType
  2. properties.category
  3. properties.marker
  4. unknown

priority 单独用于控制点大小、描边粗细和缩放显示阈值。

¶主映射

¶category 兜底

¶当前前端里的归一化别名

前端还额外处理了一些 primaryPlaceType 别名，便于命中主映射：

原值 归一化后
city gate gate
tower defensive / tower single tower
wall 2 / city wall / defensive wall wall
fort 2 fort
church 2 temple-2
church / basilica temple
shrine sanctuary
district / territory region
plain region
valley / spring / lake landscape

¶priority 样式规则

¶推荐图层策略

  • references/ancient-greece-major-leaflet.geojson：默认显示，适合放标签和主要图标
  • references/ancient-greece-leaflet.geojson：缩放较大时再显示，或做聚合
  • priority === "major"：始终显示
  • priority === "notable"：zoom >= 7
  • priority === "context"：zoom >= 9

¶当前实现说明

  • 当前前端已经按这套优先级和颜色规则渲染参考层
  • 现阶段仍使用 Leaflet circleMarker，还没有换成真正不同形状的 SVG 图标
  • 标签类名会跟随 icon 值生成，例如 hmap-ref-label--city、hmap-ref-label--temple
